Setup Guide:

Hardware Requirements:
1. Arduino Uno
2. DHT11 Temperature and Humidity Sensor
3. LCD1602 Module (with pin header)
4. 220 Ohm Resistor
5. Jumper Wires
6. Breadboard

Software Used:
1. Arduino IDE
2. DHT Sensor Library by Adafruit
3. LiquidCrystal Library
4. Baud Rate = 9600

Wiring:
1. VSS -> GND
2. VDD -> 5V
3. VO -> GND
4. RS -> Pin 12
5. RW -> GND
6. E -> Pin 11
7. D4 -> Pin 5
8. D5 -> Pin 4
9. D6 -> Pin 3
10. D7 -> Pin 7
11. A -> 5V through the 220 Ohm Resistor
12. K -> GND

13. DHT11 VCC -> 5V
14. DHT11 GND -> GND
15. DHT11 Data -> pin 2
